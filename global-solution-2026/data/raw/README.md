# data/raw — Dados Brutos

Esta pasta armazena os dados brutos utilizados nos cenários brasileiros.

## Fontes de dados

| Arquivo | Fonte | Descrição |
|---------|-------|-----------|
| municipios_rs.csv | IBGE + Defesa Civil RS | Municípios RS afetados 2024 |
| municipios_matopiba.csv | IBGE + INMET | Municípios MATOPIBA com risco de seca |
| malha_viaria_rs.geojson | DNIT | Rodovias federais RS |
| ndvi_matopiba.csv | NASA MODIS MOD13Q1 | Índice NDVI 2024 |

## Como obter os dados reais

- **IBGE**: https://ibge.gov.br/geociencias
- **DNIT**: https://dnit.gov.br
- **NASA MODIS**: https://earthdata.nasa.gov
- **INMET**: https://bdmep.inmet.gov.br
- **Defesa Civil RS**: https://defesacivil.rs.gov.br

> Os dados utilizados neste projeto são sintéticos calibrados com base nas fontes acima.
